## 📋 Sobre o Projeto

**Encantos Literários** é a landing page de um clube de assinatura fictício de livros com tema mágico e encantado. A página apresenta a proposta do clube, o kit mensal com seus itens (livro em capa dura, broche exclusivo e marca-páginas temático), os três planos de assinatura (Mensal, Semestral e Anual) com preços e benefícios, além de rodapé com links e redes sociais. O projeto foi desenvolvido com foco em **transitions** e **animations CSS**, explorando ao máximo os recursos visuais da linguagem.

## ✨ Funcionalidades

- ✅ Navbar com logo e botão de assinatura
- ✅ Hero section com headline, subtítulo e imagem temática

## 🧠 Foco Técnico — Transitions & Animations

Este projeto foi pensado como laboratório de animações e transições CSS. Alguns conceitos aplicados:

- `transition` para suavizar efeitos de hover nos cards de planos e botões de assinatura
- `@keyframes` para criar animações personalizadas nos elementos decorativos
- `animation` com `ease-in-out` e `infinite` para elementos flutuantes da seção do kit
- `transform: translateY()` e `rotate()` combinados com `animation` para movimento orgânico
- `animation-delay` para escalonar animações de múltiplos elementos
- `opacity` com `transition` para efeitos de aparição suave
